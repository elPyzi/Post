# Post project

Post is project about delivery in logistic

## Tech Stack

### Client

1) `React` + `TS`
2) `React Query` + `React Router Dom`
3) `Vite` + `Eslint` + `Prettier`

## How to start work

### Client side

1) open client package ```cd client```
2) download packages addictions ```npm i```
3) start ```npm run dev```

### Server side
